NAME
       unixterm - (simple) Remote terminal for unix sockets

SYNOPSIS
       unixterm socket

DESCRIPTION
       A unixterm is a terminal application for a unix stream socket.

       It has been created as a part of the vde-2 project: it is used as a remote console for vde_switch or for the wirefilter application.
