NAME
       vmware-toolbox-cmd - GUI toolbox (commandline version)

SYNOPSIS
       vmware-toolbox-cmd

DESCRIPTION
       This is a console version of vmware-toolbox, and depends on libvmtools and libguestlib.

OPTIONS
       vmware-toolbox-cmd has no options.

SEE ALSO
       vmware-checkvm(1)
       vmware-xferlogs(1)
       libguestlib(3)
       libvmtools(3)
       vmware-user-suid-wrapper(1)

HOMEPAGE
