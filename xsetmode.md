NAME
       xsetmode - set the mode for an X Input device

SYNOPSIS
       xsetmode device-name ABSOLUTE | RELATIVE

DESCRIPTION
       Xsetmode sets the mode of an XInput device to either absolute or relative.  This isn't appropriate for all device types.
