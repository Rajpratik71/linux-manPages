NAME
       cups-calibrate - ESP CUPS Printer Calibration Tool

SYNOPSIS
       cups-calibrate

DESCRIPTION
       This  program  allows  you  to calibrate the color output of printers using the Gutenprint CUPS
       driver. Please note that this program only works with the Gutenprint CUPS driver.

       These drivers are identified by the text CUPS+Gutenprint in the model description displayed  by
       the CUPS web interface.

       To  make  a  calibration profile for all users, run this program as the root user.  The program
       displays a series of prompts which should simply be followed.
