       chmsee â€” A chm file viewer written in GTK

SYNOPSIS
       chmsee [FILE]

       chmsee --help

       chmsee --version

DESCRIPTION
       chmsee  is a viewer for Compiled HTML Help (CHM) files. It, can show the contents tree if one is available and do the usual history stunts.
       It allows you to change fonts, use bookmarks.
