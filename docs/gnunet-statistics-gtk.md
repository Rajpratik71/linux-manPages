NAME
       gnunet-statistics-gtk - a gtk interface for showing peer statistics

SYNOPSIS
       gnunet-statistics-gtk [OPTIONS]

DESCRIPTION
       gnunet-statistics-gtk is a gtk+ based GUI for displaying selected statistics from a peer.  The
       x-axis shown is always time.

       -h, --help
              print help page

       -c FILENAME, --config=FILENAME
              load config file (default: ~/.config/gnunet.conf)

       -t, --tray
              start with main window minimized (only put icon in tray)

       -v, --version
              print the version number

FILES
       ~/.config/gnunet.conf
              User's GNUnet configuration file
