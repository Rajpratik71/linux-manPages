NAME
       stl2gts - convert an STL file to GTS format.

SYNOPSIS
       stl2gts [OPTIONS] < input.stl > output.gts

DESCRIPTION
       This manual page documents briefly the stl2gts command.

OPTIONS
       These  programs  follow  the  usual GNU command line syntax, with long options starting with two dashes (`-').  A summary of options is in‚Äê
       cluded below.

       -r, --revert
              Revert face normals.

       -n, --nomerge
              Do not merge vertices.

       -v, --verbose
              Display surface statistics.

       -h, --help
              Display the help and exit.
