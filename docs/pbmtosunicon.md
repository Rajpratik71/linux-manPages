NAME
       pbmtosunicon - convert a PBM image into a Sun icon


SYNOPSIS
       pbmtosunicon [pbmfile]


DESCRIPTION
       This program is part of Netpbm(1)

       pbmtosunicon reads a PBM image as input and produces a Sun icon as output.
