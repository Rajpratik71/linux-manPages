NAME
       ldns-chaos - give some information about a nameserver

SYNOPSIS
       ldns-chaos NAMESERVER


DESCRIPTION
       ldns-chaos retrieves all the addresses of the nameserver and then queries each address for its version.bind and hostname.bind.

       ldns-chaos is a bit more complex than ldns-mx.


OPTIONS
       ldns-chaos has no options.


AUTHOR
       Written by the ldns team as an example for ldns usage.
