NAME
       gcm-picker - GNOME Color Manager Color Picker Tool

SYNOPSIS
       gcm-picker [ --verbose ]

DESCRIPTION
       This manual page documents briefly the gcm-picker command.

       gcm-picker allows you to pick spot colors.
