NAME
       restart-dirsrv - restart script for Directory Server

SYNOPSIS
       restart-dirsrv [<instance>]

DESCRIPTION
       restart-dirsrv restarts all Directory Server instances on a system, or only a single instance if specified

OPTIONS
       <instance>
              Specifies a single Directory Server instance to restart

USAGE
       Sample usage:

       restart-dirsrv example
