NAME
       umount.ecryptfs - eCryptfs umount helper.

SYNOPSIS
       umount [ecryptfs mount point]

DESCRIPTION
       umount.ecryptfs is an eCryptfs umount helper, that will also unlink keys from the keyring.

SEE ALSO
       mount.ecryptfs(8), mount(8)

       http://ecryptfs.org/
