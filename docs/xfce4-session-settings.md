NAME
       xfce4-session-settings - Session settings for Xfce

DESCRIPTION
   Usage:
              xfce4-session-settings [OPTION...]

   Help Options:
       -?, --help
              Show help options

       --help-all
              Show all help options

       --help-gtk
              Show GTK+ Options

   Application Options:
       -s, --socket-id=SOCKET ID
              Settings manager socket

       -V, --version
              Version information

       --display=DISPLAY
              X display to use
