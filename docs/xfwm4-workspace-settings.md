NAME
       xfwm4 - Workspace settings for Xfce window manager

DESCRIPTION
   Usage:
              xfwm4-workspace-settings [OPTION...] .

   Help Options:
       -h, --help
              Show help options

       --help-all
              Show all help options

       --help-gtk
              Show GTK+ Options

   Application Options:
       -s, --socket-id=SOCKET ID
              Settings manager socket

       -V, --version
              Version information

       --display=DISPLAY
              X display to use
