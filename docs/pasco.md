NAME
       pasco - tool to extract information from MS IE cache files

SYNOPSIS
       Pasco FILE

DESCRIPTION
       pasco is a tool to extract valuable information (from a forensics investigator point of view) from MS IE cache files (index.dat).

OPTIONS
       FILE
           Cache file to parse.
