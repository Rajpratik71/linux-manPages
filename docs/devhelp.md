NAME
       devhelp - A developers help program.

SYNTAX
       devhelp [OPTIONS]

DESCRIPTION
       DevHelp is a help system for GNOME.

OPTIONS
       -s, --search=STRING
              Search for a function.

EXAMPLES
       To run this program the standard way type:

       devhelp

       If you would like to search for a function, type:

       devhelp -s search-string
