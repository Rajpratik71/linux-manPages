NAME
       scs2ps - convert IBM SCS printer data into PostScript

SYNOPSIS
       scs2ps

DESCRIPTION
       The program converts from an SCS print stream on the standard input, to a PostScriptÂ® file on the standard output. It is only intended for use with lp5250d.
