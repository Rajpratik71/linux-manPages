       SYNOPSIS

              mlxarchive OPTIONS

       DESCRIPTION

       OPTIONS

              mlxarchive OPTIONS

       -h|--help
              : Show help message and exit

       -v|--version version
              : MFA2 version in the following format: x.x.x

       -o|--out-file out_file
              : Output file

       -b|--bins-dir bins_dir
              : Directory with the binaries files
