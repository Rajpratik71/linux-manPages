NAME
       gcm-inspect - GNOME Color Manager Inspect Tool

SYNOPSIS
       gcm-inspect [ --verbose ]

DESCRIPTION
       This manual page documents briefly the gcm-inspect command.

       gcm-inspect allows you to inspect your session color management settings.
