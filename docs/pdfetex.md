NAME
       pdfetex - PDF output from e-TeX

SYNOPSIS
       pdfetex [options] [&format] [file|\commands]

DESCRIPTION
       pdfeTeX is another name for pdfTeX; they behave identically.  See pdftex(1) for all information.
