NAME
       gpgparsemail - Parse a mail message into an annotated format

SYNOPSIS
       gpgparsemail [options] [file]

DESCRIPTION
       The  gpgparsemail  is  a  utility currently only useful for debugging.
       Run it with --help for usage information.
