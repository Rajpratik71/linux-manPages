NAME
       getkeycodes - print kernel scancode-to-keycode mapping table

SYNOPSIS
       getkeycodes

DESCRIPTION
       The  getkeycodes command prints the kernel scancode-to-keycode mapping
       table.

OPTIONS
       None.

SEE ALSO
       setkeycodes (8)
