NAME
     urxvt-tabbed — urxvt with a tabbed interface

SYNOPSIS
     urxvt-tabbed [OPTIONS]

DESCRIPTION
     This is a script that starts urxvt(1) with a tabbed interface.  See urxvt(1) for a list of available options.
