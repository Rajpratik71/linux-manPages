NAME
       gpk-log - GNOME PackageKit Update Preferences

SYNOPSIS
       gpk-log [ --verbose ]

DESCRIPTION
       This manual page documents briefly the gpk-log command.

       gpk-log allows you to change preferences for keeping your system up to date.
