NAME
       btmgmt: - A command-line interface of BlueZ for management Usage:

              btmgmt [options]

OPTIONS
       --index
              Specify adapter index

       --timeout
              Timeout in seconds for non-interactive mode

       --version
              Display version

       --help Display help
