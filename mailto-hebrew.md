NAME
       mailto-hebrew - Run the mailto program to send Hebrew/English mail

SYNOPSIS
       mailto-hebrew  mailto-arguments

DESCRIPTION
       The mailto-hebrew program will start an xterm using the appropriate font in order to send mail in mixed Hebrew and English.  All comand line options will be passed on to the mailto program.

SEE ALSO
       mailto(1), shownonascii(1)

BUGS
       This is such a tiny program it might actually be bug-free, but I wouldn't bet on it.
