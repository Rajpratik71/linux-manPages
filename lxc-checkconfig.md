NAME
       lxc-checkconfig - check the current kernel for lxc support

SYNOPSIS
       lxc-checkconfig

DESCRIPTION
       lxc-checkconfig check the current kernel for lxc support

EXAMPLES
       lxc-checkconfig
              check  the current kernel.  CONFIG can be set in the envi‐
              ronment to an alternate location.
