NAME
       wxrc - command line compiler for wx xml resources.

DESCRIPTION
       The  real  documentation  for  this tool (what of it that exists) is available in the wxWidgets
       manual,  which  is  packaged  in  html  format  in  the  wx3.0-doc  package  and  installed  in
       /usr/share/doc/wx3.0-doc/html/overview_xrc.html on Debian systems.
