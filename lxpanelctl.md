NAME
       lxpanelctl - controller for lxpanel.

SYNOPSIS
       lxpanelctl {command}

DESCRIPTION
       This manual page documents briefly the lxpanelctl command.

       lxpanelctl is a program that controls lxpanel.

COMMANDS
       menu
           Show the system menu.

       run
           Show the run dialog.

       config
           Show the panel configuration dialog. If more than one panel is configured, it is undefined which panel configuration dialog will be
           shown.

       restart
           Restart lxpanel.

       exit
           Exit lxpanel.
