NAME
       uget-gtk-1to2 - Convert uGet for GTK+ data file from 1.10.x to 2.x

SYNOPSIS
       uget-gtk-1to2 [ config  directory ]

EXAMPLES
       uget-gtk-1to2  “/home/your_user/.config/uGet”
