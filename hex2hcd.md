NAME
       hex2hcd - Broadcom Bluetooth firmware converter

DESCRIPTION
       Broadcom Bluetooth firmware converter Usage:

              hex2hcd [options] <file>

OPTIONS
       -o, --output <file>
              Provide firmware output file

       -h, --help
              Show help options
