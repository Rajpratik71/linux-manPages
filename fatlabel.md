NAME
       fatlabel - set or get MS-DOS filesystem label

SYNOPSIS
       fatlabel DEVICE [LABEL]

DESCRIPTION
       fatlabel set or gets a MS-DOS filesystem label from a given device.

       If  LABEL  is  omitted,  then the label name of the specified device is written on the standard
       output.  A label can't be longer than 11 bytes.

OPTIONS
       -h, --help
           Displays a help message.

       -V, --version
           Shows version.
