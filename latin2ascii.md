NAME
       latin2ascii - converts Latin-1 characters into ASCII

SYNOPSIS
       latin2ascii [option...] file...

DESCRIPTION
       latin2ascii converts Latin-1 characters into ASCII using an in-house mapping table.

OPTIONS
       -c codec
           Specifies input file codec.

SEE ALSO
       pdf2txt(1)
