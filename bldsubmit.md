NAME
       bldsubmit - submit an IP address to a bld host

SYNOPSIS
       bldsubmit [host [port]] ip

DESCRIPTION
       bldsubmit submits ip to the bld daemon listening on host:port (default localhost:2905).

       bld(8) bldread(8) bldquery(8) bldinsert(8) blddecr(8)
