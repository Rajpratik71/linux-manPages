NAME
       one - OpenNebula startup script

SYNOPSIS
       one <COMMAND>

DESCRIPTION
       one start or stop OpenNebula core and scheduler

COMMANDS
       start

              start the oned and mm_sched services

       stop

              stop the oned and mm_sched services
